## 📖 프로젝트 소개

웹 표준과 접근성을 고려한 코드 작성을 지향하며, 지속적인 학습과 반복적인 개선을 통해 더 나은 구조와 품질의 코드를 만들어가는 UI Developer 이원경의 포트폴리오 사이트입니다.

### ✨ 주요 특징

- **🎨 모던한 UI/UX**: 직관적이고 세련된 디자인
- **📱 완전 반응형**: 모든 디바이스에서 최적화된 경험
- **🎭 인터랙티브 애니메이션**: AOS와 CSS 애니메이션으로 부드러운 사용자 경험
- **⚡ 최적화된 성능**: Vite를 활용한 빠른 로딩과 최적화된 번들링
- **🎯 접근성**: 웹 표준과 접근성 가이드라인 준수

## 🛠️ 기술 스택

### Frontend
- **React 19.1.1** - 컴포넌트 기반 UI 라이브러리
- **Vite 7.1.2** - 차세대 프론트엔드 빌드 툴
- **SCSS** - CSS 전처리기로 체계적인 스타일 관리
- **JavaScript (ES6+)** - 모던 자바스크립트

### 라이브러리 & 도구
- **AOS** - 스크롤 애니메이션
- **Swiper** - 터치 슬라이더
- **React Scroll** - 부드러운 페이지 스크롤
- **ESLint** - 코드 품질 관리

### 배포 & 인프라
- **Vercel** - 자동화된 배포 및 호스팅
- **Git & GitHub** - 버전 관리

## 📁 프로젝트 구조

```
portfolio/
├── public/
│   └── images/           # 이미지 에셋
│       ├── icon_*.png    # 스킬 아이콘
│       ├── work_*.png    # 프로젝트 이미지
│       └── ...
├── src/
│   ├── Components/       # React 컴포넌트
│   │   ├── About.jsx     # 소개 섹션
│   │   ├── Contact.jsx   # 연락처 및 스킬 섹션
│   │   ├── Footer.jsx    # 푸터
│   │   ├── Header.jsx    # 네비게이션
│   │   ├── Main.jsx      # 메인 히어로 섹션
│   │   ├── Projects.jsx  # 프로젝트 포트폴리오
│   │   └── styles/       # SCSS 스타일 모듈
│   │       ├── _variables.scss  # 글로벌 변수
│   │       └── *.module.scss    # 컴포넌트별 스타일
│   ├── config.js         # 프로젝트 및 스킬 데이터
│   ├── App.jsx           # 루트 컴포넌트
│   └── main.jsx          # 엔트리 포인트
├── package.json
└── vite.config.js        # Vite 설정
```

## 🎯 주요 기능

### 📱 섹션별 기능

1. **Header** - 부드러운 스크롤 네비게이션
2. **Main** - 패럴랙스 효과가 적용된 히어로 섹션
3. **About** - 경력사항, 자격증, 학력 정보
4. **Projects** - 인터랙티브 프로젝트 슬라이더
5. **Contact** - 무한 스크롤 스킬 배지와 연락처 정보

### 🎨 디자인 시스템

- **반응형 클램프 함수** 사용으로 모든 디바이스 대응
- **CSS 커스텀 속성**을 활용한 일관된 색상 시스템
- **모듈화된 SCSS** 구조로 유지보수성 향상
- **접근성** 고려한 색상 대비와 키보드 네비게이션

## 🚀 시작하기

### 필수 조건

- Node.js 18.0.0 이상
- npm 또는 yarn

### 설치 및 실행

1. **저장소 클론**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **의존성 설치**
   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

4. **빌드**
   ```bash
   npm run build
   ```

5. **프리뷰**
   ```bash
   npm run preview
   ```

## 📊 성능 최적화

- **이미지 최적화**: Vite의 에셋 처리로 자동 압축 및 캐시 버스팅
- **코드 스플리팅**: 컴포넌트별 모듈화로 효율적인 번들 크기
- **CSS 최적화**: SCSS 변수와 믹스인으로 중복 코드 제거
- **라이브러리 최적화**: 필요한 기능만 import하여 번들 크기 최소화
